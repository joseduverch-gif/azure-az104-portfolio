\# Proyecto 19 - Microsoft Defender for Cloud



\## Objetivo



Explorar Microsoft Defender for Cloud para evaluar la postura de seguridad de la suscripción de Azure, revisar el Secure Score y analizar las recomendaciones de seguridad generadas automáticamente para los recursos implementados.



\---



\## Arquitectura



Microsoft Defender for Cloud proporciona:



\- Evaluación continua de la postura de seguridad.

\- Secure Score.

\- Recomendaciones para mejorar la seguridad.

\- Protección para máquinas virtuales, almacenamiento, Key Vault y otros servicios de Azure.



\---



\## Recursos evaluados



\- Virtual Machine

\- Storage Account

\- Azure Key Vault

\- Resource Manager

\- Suscripción de Azure

\## Resultados obtenidos



Durante el laboratorio se verificó que Microsoft Defender for Cloud:



\- Detecta configuraciones inseguras.

\- Calcula el Secure Score de la suscripción.

\- Genera recomendaciones específicas para cada recurso.

\- Prioriza las acciones para fortalecer la postura de seguridad.



Entre las recomendaciones observadas se encontraron:



\- Configuración de Azure Key Vault.

\- Protección de Storage Account.

\- Evaluación de vulnerabilidades de máquinas virtuales.

\- Configuración de Microsoft Defender para distintos servicios de Azure.



\---



\## Conclusiones



Microsoft Defender for Cloud permite supervisar continuamente la seguridad de una suscripción de Azure y proporciona recomendaciones para reducir riesgos, mejorar el cumplimiento y fortalecer la infraestructura mediante buenas prácticas de seguridad.
