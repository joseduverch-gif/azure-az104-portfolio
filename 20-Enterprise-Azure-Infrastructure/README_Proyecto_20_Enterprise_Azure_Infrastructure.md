# Proyecto Final - Infraestructura Empresarial en Microsoft Azure

## Descripción

Este proyecto integra los principales servicios implementados durante el
laboratorio AZ-104 en una única solución empresarial.

## Objetivos

-   Implementar una infraestructura empresarial.
-   Integrar redes, seguridad, monitoreo y respaldo.
-   Demostrar competencias de Azure Administrator.

## Escenario

Contoso Solutions migra su infraestructura a Microsoft Azure.

## Componentes

### Redes

VNET, NSG, Route Table, VPN Gateway y Load Balancer.

### Cómputo

VM y VM Scale Set.

### Seguridad

Key Vault, Microsoft Entra ID, RBAC y Defender for Cloud.

### Monitoreo

Azure Monitor con Alert Rule y Action Group.

### Backup

Recovery Services Vault.

## Validaciones

-   VM operativa.
-   VPN Gateway implementado.
-   Monitor y Alertas configuradas.
-   Backup configurado.
-   Key Vault implementado.
-   Defender for Cloud analizando la suscripción.

## Buenas prácticas

-   Uso de Resource Group.
-   Protección mediante NSG.
-   Secretos en Key Vault.
-   Monitoreo continuo.
-   Backups.

## Tecnologías

Azure VM, VNet, NSG, VPN Gateway, Load Balancer, Storage Account, Key
Vault, Azure Monitor, Azure Backup, Defender for Cloud, Microsoft Entra
ID.

## Conclusión

La solución integra redes, cómputo, almacenamiento, monitoreo, respaldo
e identidad, demostrando capacidades prácticas alineadas con AZ-104.
